# رِقَاب (Reqab)

**عينٌ خبيرة على كل بند.**

نموذج أولي لهاكاثون أمد 2026 (بالشراكة بين مصرف الإنماء وأكاديمية طويق)، مسار الذكاء الاصطناعي التوليدي للتقنية المالية.

رِقَاب نظام ذكاء اصطناعي توليدي لعقود التمويل: يقرأ العقد (نصًا أو PDF أو صورة) ويكشف فورًا الرسوم الخفية والبنود الخطرة والمخالفات الشرعية والنظامية (ضوابط المرابحة والتورق ومبادئ ساما)، ويولّد مسودات عقود تمويل متوافقة من بيانات العميل.

**تجربة مباشرة:** https://reqab.vercel.app

## المزايا

- **تحليل عقد**: بطاقة درجة أمان من 100، جدول الأرقام الحقيقية (التكلفة الفعلية الكاملة)، بطاقات البنود الخطرة مع اقتباس النص الأصلي وأثره المالي، فحص شرعي ونظامي بند ببند، وتوصيات قبل التوقيع.
- **توليد مسودة عقد**: مرابحة أو تورق كاملة البنود متوافقة شرعًا ونظامًا من بيانات العميل، تُعرض في بطاقة عقد رسمية وتُحمَّل ملف PDF بترويسة رِقَاب وجدول توقيعات.
- **واجهة تفاعلية**: ردود الذكاء الاصطناعي تُعرض بمكوّنات مرئية (مقياس درجة الأمان، بطاقات المخاطر، أسطر الفحص) وليس نصًا فقط.
- **إرفاق مستندات**: PDF وصور ونصوص تُقرأ مباشرة عبر Gemini.
- **استرجاع معزز (RAG) من الأنظمة الرسمية**: مكتبة مفهرسة عبر Gemini File Search مصادرها حكومية حصريًا (.gov.sa): من البنك المركزي السعودي (rulebook.sama.gov.sa) ضوابط التمويل الاستهلاكي المحدثة ومبادئ حماية عملاء شركات التمويل واللائحة التنفيذية لنظام مراقبة شركات التمويل ومبادئ حماية عملاء المؤسسات المالية، ومن هيئة الخبراء بمجلس الوزراء (laws.boe.gov.sa) نظام مراقبة شركات التمويل ونظام التمويل العقاري؛ يستند إليها الفحص ويعرض المصادر أسفل كل تقرير.
- **عقد تجريبي مدمج**: عقد مرابحة واقعي مزروعة فيه 9 مخاطر خفية لعرض القدرات فورًا.

## التقنيات

Next.js (App Router) · Vercel AI SDK v7 · Google Gemini (gemini-3.5-flash) · Gemini File Search (RAG) · shadcn/ui · Tailwind CSS v4 · Streamdown

## التشغيل محليًا

```bash
npm install
echo "GEMINI_API_KEY=<your-key>" > .env
npm run dev
```

ثم افتح http://localhost:3000

لتفعيل الاسترجاع من الأنظمة (اختياري): شغّل `node --env-file=.env scripts/seed-file-search.mjs` لفهرسة مستندات مجلد `corpus/` في مخزن Gemini File Search، ثم أضف الناتج `FILE_SEARCH_STORE_NAME=...` إلى `.env`. يعمل التطبيق بدون هذه الخطوة لكن دون استشهاد بالمصادر.
