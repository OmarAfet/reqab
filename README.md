# رِقَاب (Reqab)

**عينٌ خبيرة على كل بند.**

نموذج أولي لهاكاثون أمد 2026 (بالشراكة بين مصرف الإنماء وأكاديمية طويق)، مسار الذكاء الاصطناعي التوليدي للتقنية المالية.

رِقَاب نظام ذكاء اصطناعي توليدي لعقود التمويل: يقرأ العقد (نصًا أو PDF أو صورة) ويكشف فورًا الرسوم الخفية والبنود الخطرة والمخالفات الشرعية والنظامية (ضوابط المرابحة والتورق ومبادئ ساما)، ويولّد مسودات عقود تمويل متوافقة من بيانات العميل.

**تجربة مباشرة:** https://reqab.vercel.app

## المزايا

- **تحليل عقد**: بطاقة درجة أمان من 100، جدول الأرقام الحقيقية (التكلفة الفعلية الكاملة)، بطاقات البنود الخطرة مع اقتباس النص الأصلي وأثره المالي، فحص شرعي ونظامي بند ببند، وتوصيات قبل التوقيع.
- **توليد مسودة عقد**: مرابحة أو تورق كاملة البنود متوافقة شرعًا ونظامًا من بيانات العميل.
- **واجهة تفاعلية**: ردود الذكاء الاصطناعي تُعرض بمكوّنات مرئية (مقياس درجة الأمان، بطاقات المخاطر، أسطر الفحص) وليس نصًا فقط.
- **إرفاق مستندات**: PDF وصور ونصوص تُقرأ مباشرة عبر Gemini.
- **عقد تجريبي مدمج**: عقد مرابحة واقعي مزروعة فيه 9 مخاطر خفية لعرض القدرات فورًا.

## التقنيات

Next.js (App Router) · Vercel AI SDK v7 · Google Gemini (gemini-3.5-flash) · shadcn/ui · Tailwind CSS v4 · Streamdown

## التشغيل محليًا

```bash
npm install
echo "GEMINI_API_KEY=<your-key>" > .env
npm run dev
```

ثم افتح http://localhost:3000
